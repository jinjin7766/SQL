## 프로젝트 요약 
Oracle SQL을 활용해 저출산의 주요 원인들을 분석하고, 데이터 기반으로 정책적 개선 방향을 제안했습니다.


## ✅진행 배경 
- **목표 및 방향성**: 출산율이 지속적으로 감소하면서 국가적, 사회적 문제가 대두되었습니다. 경제적 부담, 주거 환경, 취업 여건, 사회적 역할 변화 등 다양한 요인이 출산율 감소에 영향을 미치고 있으며, 이를 정량적으로 분석하여 정책적 개선 방향을 제안하고자 프로젝트를 진행했습니다.


## 📊성과 및 결과  
- **결과물 및 성과**: 사업장 규모가 작을수록 출산율 감소폭이 크고, 여성 경제활동 참여 증가는 출산율 감소, 최종 학력 수준이 높아질수록 미혼율이 증가하고, 출산율 감소폭이 큰 지역에서는 주거비 부담 증가와 연관성이 확인되었으며, 이를 바탕으로 경제적 부담 완화, 여성의 사회적 참여 증진, 주거 환경 개선을 통한 출산 장려 방안을 제안했습니다.

 
## 🛠사용 언어와 라이브러리 
- **프로그래밍 언어**: Oracle SQL
- **주요 데이터 항목**: 경제적 부담(사업장 규모별 출산율), 사회적 역할 변화(최종 학력), 취업 여건(여성 경제활동 참가율), 주거 환경(부동산 소비 심리지수)

## 📌프로젝트 진행 과정 
- **작업 단계**: 경제적 부담, 사회적 역할 변화, 취업 여건, 주거 환경 관련 데이터 수집 -> SQL을 활용해 데이터 테이블 생성 및 변수 정리 -> 사업장 규모별 출산율 변화율 분석, 지역별 부동산 소비 심리지수와 출산율 감소폭 비교, 연령별/학력별 출산율 변화 추적


## ❗️한계점 및 어려웠던 점 
- **제약 조건**: 주거 환경)와 출산율 간의 인과 관계를 정밀히 검증하지 못함
- **해결 방법**: 추가 데이터를 활용해 분석 결과를 보완하고 다양한 요인을 교차 분석하여 신뢰도를 높였습니다
- **아쉬운점**: 더욱 심층적인 정책 제안을 위해 실질적인 정책 데이터를 추가로 확보하지 못한 점

## 💻학습 및 성장 
- **얻은 교훈**: 데이터 기반의 정책 제안 시, 단순한 상관 관계를 넘어 더 깊은 인과 관계를 도출해야 한다는 필요성을 느꼈습니다
- **기술 향상**: SQL을 활용한 데이터 정리 및 분석 능력 향상, 데이터 기반으로 인사이트를 도출하는 능력 강화
- **개인적/팀워크 성장**: 팀원들과의 협업을 통해 데이터 분석과 정책 제안에 필요한 다양한 관점을 수용하며 시야를 넓혔습니다.
